Single Header File, Single C File

Peppa PEG is a small library in about 2k LOC. Want to use it? Copy peppapeg.h and peppapeg.c to the root directory of your project!

Full PEG Implementation

You can enjoy the full PEG elements, such as literal, range, positive, negative, reference, sequence, choice, repetition, etc. In addition, Peppa PEG provides some extensions for altering the generated AST, such as flags, callbacks, etc.

Easy to Use PEG & Low-Level APIs

Keep it simple, stupid! The API of Peppa PEG is aimed to be as simple as possible. Read the documentation here: Documentation.
